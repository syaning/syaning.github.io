# 709. To Lower Case

https://leetcode.com/problems/to-lower-case/

## JavaScript

```js
/**
 * @param {string} str
 * @return {string}
 */
var toLowerCase = function(str) {
  return str.toLowerCase()
}
```

## Python

```py
class Solution:

    def toLowerCase(self, str):
        """
        :type str: str
        :rtype: str
        """
        return str.lower()
```
