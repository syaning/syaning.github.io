# 数据类型

## 标量类型

### 整数

- Signed: `i8`, `i16`, `i32`, `i64`, `i128`, `isize`
- Unsigned: `u8`, `u16`, `u32`, `u64`, `u128`, `usize`

默认为 `i32`。

字面量表示参考如下：

- 十进制：`98_222`
- 十六进制：`0xff`
- 八进制：`0o77`
- 二进制：`0b1111_0000`
- Byte(`u8`类型)：`b'A'`

其中 `_` 是一个虚拟的分隔符，用于提高可读性。例如 `98_222` 和 `98222` 是等同的。

### 浮点数

`f32` 和 `f64`，默认为 `f64`。

### bool

类型为 `bool`，取值为 `true` 和 `false`。

### char

类型为 `char`，大小为四个字节。单引号表示，例如 `'a'`，`'好'`。

## 复合类型

### tuple

- 长度固定
- 元素的数据类型可以不同
- 支持解构操作
- 支持 `data.index` 方式访问
- 单个元素的 tuple，最后必须有一个逗号，例如
	- `(1,)` 是一个 tuple
	- `(1)` 表示数字 1

```rust
fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);

    let (x, y, z) = tup;

    println!("{} {} {}", x, y, z);

    println!("{} {} {}", tup.0, tup.1, tup.2)
}
```

### array

- 长度固定
- 元素的数据类型必须相同
- 通过 `data[index]` 方式访问
- `.len()` 获取数组长度

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];
    println!("{:?}", a); // [1, 2, 3, 4, 5]

    let b: [i32; 5] = [1, 2, 3, 4, 5];
    println!("{:?}", b); // [1, 2, 3, 4, 5]

    let c = [3; 5];
    println!("{:?}", c);     // [3, 3, 3, 3, 3]
    println!("{}", c[0]);    // 3
    println!("{}", c.len()); // 5
}
```

## 特殊类型

没有任何元素的 tuple 为 `()`，它是一个特殊类型 (unit type)，其唯一取值也是 `()`(unit value)。
